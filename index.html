JavaScriptning Function mavzusi

Function dasturning ma'lum bir bajaradigon bo'lag va functionlar qayta-qayta ishlaydi.
Function 3ga bo'linadi

1. Function Decloration

2. Function Expression

3. Arrow Function

1. Function Decloration misol keltiraman
Function sayHello(){
console.log('Hello world')
}
sayHello()
qoida: Function Declorationda sayHello() ni tepada yozsa xam ishlaydi.
2. Function Expression misol keltiraman
let sayByeBye = function(){
console.log('sayByeBye function expression')
}
sayByeBye()
qoida: Function Expressionda sayByeBye() ni tepada yozsa ishlamaydi.
3. Arrow function misol keltiraman
let nico = () => {
console.log(' This is arrow function')
}
nico()
qoida: Arrowda function ishlamaydi.

JavaScriptning Date mavzusi

Date bizga yilni, oyni, kuni, va soat vaqtini aniqlab beradi.

Shularga bir nechta misollar keltiraman.

let now = newDate()
let today = now.getDate() < 10 ? '0' + now.getDate() : now.getDate()
let month = now.getMonth() < 10 ? '0' + now.getMonth() : now.getMonth()
let year = now.getFullYear()
console.log('Today:', today)
console.log('Month:', month)
console.log('Year:', year)

Date mavsusida soatni yasash

let hour = now.getHours() < 10 ? '0' + now.getHours() : now.getHours()
let minute = now.getMinutes() < 10 ? '0' + now.getMinutes() : now.getMinutes()
let second = now.getSeconds() < 10 ? '0' + now.getSeconds() : now.getSeconds()
console.log(`${hour}:${minute}:${second}`)
